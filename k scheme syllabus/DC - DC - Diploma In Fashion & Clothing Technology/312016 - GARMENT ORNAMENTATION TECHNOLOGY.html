I. RATIONALE

Garment ornamentation is crucial for value addition. This course provides hands-on experience to the students to use different techniques that are useful to create texture and richly embellished fabric and accessories. This course is useful for the garment industry, Startups/Employment. It also helps to add splendor to clothing using decorative methods of handicrafts, yarn crafts, adornments, etc.

II. INDUSTRY / EMPLOYER EXPECTED OUTCOME
The aim of this course is to help the student to attain the industry expected outcome through various teaching-learning experiences: Apply relevant garment ornamentation techniques in various situations.
III. COURSE LEVEL LEARNING OUTCOMES (COS)
Students will be able to achieve & demonstrate the following COs on completion of course based learning
  • CO1 - Use relevant tools and accessories for garment ornamentation.
  • CO2 - Use different types of basic hand embroidery stitches and thread work.
  • CO3 - Identify different Surface ornamentation techniques for garment value addition.
  • CO4 - Use different fabric manipulation techniques.
