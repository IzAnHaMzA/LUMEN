I. RATIONALE

The garment industry uses CAD/CAM software to help designers work faster, improve design quality, and enhance communication and manufacturing processes. This course enables students to create unique weaves, prints, and designs, and learn to draft, modify, grade, and plan markers for effective manufacturing. Hands-on experience with CAD/CAM software enhances their design skills and efficiency.

II. INDUSTRY / EMPLOYER EXPECTED OUTCOME
This course aims to help the student attain the industry's expected outcome through various teaching-learning experiences: Use relevant CAD/CAM software for textile and apparel designing and manufacturing.
III. COURSE LEVEL LEARNING OUTCOMES (COS)
Students will be able to achieve & demonstrate the following COs on completion of course based learning
  • CO1 - Apply CAD tools to create intricate motifs for weaving and printing on fabrics.
  • CO2 - Design various garment styles using relevant CAD software.
  • CO3 - Draft and grade patterns using various CAD software.
  • CO4 - Prepare marker plans for various patterns using computerized marker making system.
  • CO5 - Select styles, fabrics, and colours for various garments using CAD Software (Q3D,Tailor I and Try ON software).
