I. RATIONALE

Oil hydraulic systems and pneumatic systems are widely used in all fields of polymer engineering as clean source of motive power by Polymer technocrats. Low cost automation systems with the use of pneumatic have become popular as manufacturing aids. This course focuses on the skills and insights of oil hydraulics and pneumatic systems for Polymer processing plants which will be directly needed in the industries.

II. INDUSTRY / EMPLOYER EXPECTED OUTCOME
The aim of this course is to help the student to attain the following industry identified competency through various teaching learning experiences: Identify Hydraulic and Pneumatic components in polymer processing unit, Select Various Hydraulic and Pneumatic components for given polymer processing machine.
III. COURSE LEVEL LEARNING OUTCOMES (COS)
Students will be able to achieve & demonstrate the following COs on completion of course based learning
  • CO1 - Identify fluid in a given power system
  • CO2 - Identify components of hydraulic and pneumatic system in given polymer processing
  • CO3 - Select pump, actuators, compressor for given fluid operated system
  • CO4 - Select appropriate control valve for given fluid operated system
  • CO5 - Select appropriate accessories for given fluid operated system
