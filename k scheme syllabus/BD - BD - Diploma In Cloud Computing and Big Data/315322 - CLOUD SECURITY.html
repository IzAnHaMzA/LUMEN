I. RATIONALE

Cloud Security is crucial because of the rapid expansion of Cloud Technology and changing threats. This course provides practical, hands-on training to help individuals build careers in Cloud Security by teaching them how to design, monitor and manage secure Cloud Systems

II. INDUSTRY / EMPLOYER EXPECTED OUTCOME
The aim of this course is to help the student to attain the following industry identified competency through various teaching learning experiences: Implement, manage and monitor Cloud Security.
III. COURSE LEVEL LEARNING OUTCOMES (COS)
Students will be able to achieve & demonstrate the following COs on completion of course based learning
  • CO1 - Identify and Mitigate Cloud-Specific Vulnerabilities
  • CO2 - Implement Infrastructure Security at different levels.
  • CO3 - Apply the concept of Data Security and Data Storage.
  • CO4 - Evaluate IAM Standards and Protocols
  • CO5 - Mange security in Cloud.
