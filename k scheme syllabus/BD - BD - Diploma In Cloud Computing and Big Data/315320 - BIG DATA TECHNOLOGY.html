I. RATIONALE

Big Data Technology encompasses the software tools used to manage various types of datasets and convert them into actionable insights for businesses. This technology enables the analysis, processing, and extraction of valuable information from large,complex datasets. This course is designed to equip students with the knowledge and skills necessary to effectively work with Big Data Technologies.

II. INDUSTRY / EMPLOYER EXPECTED OUTCOME
Use Big Data Technologies to manage, analyze and derive insights from large volumes of data effectively.
III. COURSE LEVEL LEARNING OUTCOMES (COS)
Students will be able to achieve & demonstrate the following COs on completion of course based learning
  • CO1 - Illustrate characteristics and technologies of Big Data.
  • CO2 - Apply relevant Big Data Storage techniques for a given application.
  • CO3 - Use Big Data Technologies Hive and Pig to handle different nature of data.
  • CO4 - Implement Big Data Processing Technique for a given application.
  • CO5 - Apply Data Visualization Techniques for a given application.
