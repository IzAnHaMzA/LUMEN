I. RATIONALE

Industrial Automation is the method to improve quality and flexibility in manufacturing system by enhancing safety with reduced cost. To achieve this, industries are shifting towards modern automation technologies. Learners of this course will become well-competent in designing, maintaining, and operating automation systems for controlling various processes using PLC technology.

II. INDUSTRY / EMPLOYER EXPECTED OUTCOME
The aim of this course is to help students to attain the following industry , employer expected outcome through various teaching learning experiences.
"To troubleshoot the process automation system , as well as design , interface and program the PLC-based industrial applications."
III. COURSE LEVEL LEARNING OUTCOMES (COS)
Students will be able to achieve & demonstrate the following COs on completion of course based learning
  • CO1 - Describe basics of Industrial Automation system and its components.
  • CO2 - Interface I/O devices with appropriate I/O modules of PLC.
  • CO3 - Test Ladder logic program for given logic gates and Boolean equations.
  • CO4 - Test ladder logic program for a given industrial application based on process statement.
  • CO5 - Use of various industrial automation system communication protocols.
