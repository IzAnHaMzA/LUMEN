I. RATIONALE

The mining industry is increasingly embracing advanced technology to improve efficiency, safety, and environmental sustainability. Information Technology (IT) plays a pivotal role in transforming traditional mining operations by introducing sophisticated tools and software for various applications such as surface mining, mine surveying, mine planning, and project monitoring. This curriculum aims to equip diploma students with the necessary knowledge and skills to leverage IT in modern mining practices, ensuring they are well-prepared for the evolving demands of the industry.

II. INDUSTRY / EMPLOYER EXPECTED OUTCOME
Technical Proficiency: Employers expect graduates to be proficient in using various IT tools and software pertinent to mining operations. Problem-Solving Skills: The ability to analyze data, identify issues, and develop IT-based solutions is highly valued. Adaptability and Continuous Learning: The fast-paced evolution of technology necessitates a mindset of continuous learning and adaptation. Safety and Environmental Awareness: Knowledge of how IT can improve safety and reduce environmental im30pact is crucial.
III. COURSE LEVEL LEARNING OUTCOMES (COS)
Students will be able to achieve & demonstrate the following COs on completion of course based learning
  • CO1 - Understand the basic concepts and importance of IT in mining operations.
  • CO2 - Use IT tools to optimize surface mining activities and improve mapping accuracy.
  • CO3 - Apply IT in mine surveying to collect and analyze precise data for mine mapping.
  • CO4 - Utilize mine planning software to design and schedule efficient mining operations.
  • CO5 - Implement IT solutions for effective project monitoring and management in mining.

VII. SUGGESTED MICRO PROJECT / ASSIGNMENT/ ACTIVITIES FOR SPECIFIC LEARNING / SKILLS DEVELOPMENT (SELF LEARNING)
Micro project
  • Project Title: Mining Site Spatial Data Analysis Using Excel Objective: Import GIS data into Excel and create pivot tables and charts to analyze the distribution of mineral resources across a mining site.
  • Project Title: Haul Road Design Optimization Using Excel. Objective: Develop a model in Excel to calculate the most efficient haul road layout for a surface mining operation. Use formulas to optimize distance and cost.
  • Prepare a microproject on different IT tools and their roles in mining (e.g., GIS, GPS, mine planning software).
  • Use a software tool like MineSight or Deswik (or any other) to simulate a basic mining operation.
  • Prepare a microproject on the use of remote sensing and satellite imagery for mining exploration and monitoring.
  • Use Excel to create a cost-benefit analysis of integrating IT in mining operations (e.g., automation, monitoring systems).
  • Conduct a survey (or use secondary data) on how mining companies are adopting IT, focusing on benefits (e.g., efficiency, safety) and challenges (e.g., cost, technical complexity).
  • Prepare a case study or a report on how safety has improved with the integration of IT tools in mining environments.
  • Prepare a presentation on the latest advancements in smart mining technologies, such as the use of drones, AI, or IoT.
  • Create a short prototype or report explaining how VR/AR technologies can improve safety training and operational efficiency in the mining industry.
  • Analyze a real-world case where IT was implemented in mining operations. Include benefits like improved safety, productivity, and cost reduction, as well as challenges like high implementation costs or data security.
  • Use Excel to compare efficiency metrics (e.g., cost, time, safety incidents) between traditional mining methods and IT-enhanced approaches. Create graphs and charts to visualize data.
  • Using flowchart software, students will design a conceptual workflow showing how IT tools (e.g., sensors, software) can be integrated into various stages of mining, from exploration to closure.
  • Any other topic related to Laboratory learning outcome and aligned practical / Tutorial experiences.
